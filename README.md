# Personal Introduction Program

## Project Overview
This is a beginner-friendly Python project that collects user information using input() and displays a personalized welcome message.

## Features
- Takes user name as input
- Takes age as input
- Takes hobby as input
- Takes city as input
- Displays friendly welcome message

## Technologies Used
- Python
